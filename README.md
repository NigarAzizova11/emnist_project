#  EMNIST Handwritten Character Recognition

This project is a web-based application that recognizes handwritten digits and letters using a deep learning model trained on the EMNIST dataset.

##  Features
- Recognizes digits (0-9), uppercase (A-Z), and lowercase (a-z)
- Real-time prediction from user drawing
- Image preprocessing (cropping, resizing, normalization)
- Confidence score for predictions
- Simple and interactive UI

##  Technologies Used
- Python
- Flask
- TensorFlow / Keras
- NumPy
- Pillow (PIL)
- HTML, CSS, JavaScript
